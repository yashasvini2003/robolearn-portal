# RoboLearn: A Beginner Robotics and Coding Learning Portal

RoboLearn is a beginner-friendly learning portal designed to introduce newcomers to the fundamentals of robotics and coding. It presents concepts in a clear, structured way through short lessons, simple examples, and interactive features. The goal is to demonstrate how technical concepts can be explained in an accessible manner suitable for students encountering robotics for the first time.

---

## Purpose of the Project

The main motivation behind this project was to combine my interests in software development, robotics, and teaching. While building RoboLearn, I aimed to strengthen my front-end development skills and practice presenting technical information in a way that beginners can understand.

The project reflects the type of content used in robotics workshops, school clubs, and introductory STEM programs. It also serves as a demonstration of how web technologies can be used to support learning.

---

## Project Overview

RoboLearn is a static website created using HTML, CSS, and JavaScript. It includes:

- Four introductory lessons on robotics fundamentals.
- Basic Arduino-style pseudocode to help bridge theory and application.
- An interactive quiz for reinforcing concepts.
- A troubleshooting section that explains common beginner issues.
- A brief guide intended for tutors or workshop leaders.

The emphasis throughout the site is clarity, simplicity, and a smooth learning experience.

## How to Run the Project

RoboLearn does not require any installation or build tools. It can be run directly in a browser.

### Method 1: Open Directly

1. Download or clone the repository.
2. Navigate to the project folder.
3. Open the `index.html` file in any web browser.

### Method 2: Using Visual Studio Code (Recommended)

1. Open the project folder in Visual Studio Code.
2. Install the "Live Server" extension.
3. Right-click on `index.html` and select "Open with Live Server".

## What This Project Covers

### Introductory Lessons

The lessons provide a structured introduction to:

- What robots are and how they are used  
- Common sensors and motors  
- Simple robot decision-making logic  
- A conceptual mini-project involving a line-following robot  

The content is written for beginners, focusing on clarity rather than technical depth.

### Interactive Quiz

A short quiz, built with JavaScript, helps learners check their understanding of the main ideas covered in the lessons.

### Troubleshooting Guidance

The troubleshooting section covers common issues beginners face when working with robotics, along with straightforward explanations to help develop problem-solving skills.

### Tutor Support

A small guide explains how tutors or workshop facilitators can use RoboLearn as a simple curriculum for teaching basic robotics concepts.
